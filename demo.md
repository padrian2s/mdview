# Welcome to Glow

A beautiful terminal Markdown viewer.

## Features

- **Clean output** - Minimal and readable
- **Syntax highlighting** - Code blocks stand out
- **GFM support** - GitHub Flavored Markdown
- **Pipe friendly** - Works with stdin

## Installation

```bash
npm install
node glow.js README.md
```

## Code Example

```javascript
function greet(name) {
    return `Hello, ${name}!`;
}

console.log(greet('World'));
```
